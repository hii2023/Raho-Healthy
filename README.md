# Raho Healthy — Personalized Diet Plan Quiz

An interactive health questionnaire that generates a **personalized diet plan recommendation** based on the user's health condition. Branded by NOSH7 as a lead-generation tool.

## What It Does

Users answer questions about their health condition (PCOD, diabetes, lifestyle), and the tool instantly recommends a tailored diet plan — then nudges them toward NOSH7's healthy meal subscription.

## Key Features

- **Multi-step health quiz** — question-by-question disease/condition screener
- **Personalized diet plan generator** — plan varies based on answers (PCOD, diabetes, weight loss, general health)
- **Multi-language support** — i18n via `data-i18n` attributes for Hindi/English switching
- **Shareable plan card** — users can save or share their recommended plan as an image
- **NOSH7 promotional CTA** — converts health-conscious visitors into subscribers
- **Sticky bottom action bar** — CTA always visible while scrolling
- **Medical disclaimer footer** — responsible health messaging
- **Single-file, no build step** — runs entirely in the browser

## Tech Stack

- HTML, Tailwind CSS (CDN), Vanilla JavaScript
- i18n via `data-i18n` attributes

## Audience

People in India dealing with PCOD, diabetes, or lifestyle health conditions who are looking for a diet solution — and potential NOSH7 subscribers.
